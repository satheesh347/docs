# **How to Use Waves Text-to-Speech (TTS)**

Waves lets you convert any text into **hyper realistic speech in under 100 ms**.\
This quick guide shows you how to make your first TTS request using cURL, Python, or JavaScript.

## **1. Get Your API Key**

- Log in to your Waves dashboard → open the **API Key** tab → copy your key.
- For security, store it in an environment variable:

```bash
export SMALLEST_API_KEY="your_api_key_here"
```

## **2. API Endpoint**

- This is the WebSocket URL where you connect to start receiving streamed audio.

```bash
https://waves-api.smallest.ai/api/v1/lightning/get_speech
```

## **3. Required Headers**

- Headers are **extra information** you send along with your API request so the server can understand who you are and what you want.
- These headers must be included during the WebSocket upgrade request :

```bash
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

## **4. Make Your First Request**

```python
##Python

from smallestai.waves import WavesClient

client = WavesClient(api_key="YOUR_API_KEY")

client.synthesize(
    text="Welcome to Waves, the fastest text to speech platform.",
    voice_id="default",
    sample_rate=24000,
    speed=1.0,
    save_as="speech.wav"
)
```

```javascript
//JavaScript

const fs = require('fs');

const response = await fetch(
  'https://waves-api.smallest.ai/api/v1/lightning/get_speech',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.SMALLEST_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      text: 'Welcome to Waves, the fastest text to speech platform.',
      voice_id: 'default',
      sample_rate: 24000,
      speed: 1.0
    })
  }
);

const buffer = await response.arrayBuffer();
fs.writeFileSync('speech.wav', Buffer.from(buffer));
```

## **5. Response Format**

- **Status:** `200 OK`
- **Content-Type:** `audio/wav`
- **Format:** PCM WAV, mono
- The body contains the **raw audio bytes** you can save as a `.wav` file.

## **6. Key Parameters**

| Parameter     | Type   | Default     | What it Does                                       |
| :------------ | :----- | :---------- | :------------------------------------------------- |
| `text`        | string | --          | Text you want to convert (recommended ≤500 chars)  |
| `voice_id`    | string | `"default"` | Choose a built-in or custom cloned voice           |
| `sample_rate` | int    | `24000`     | 8k / 16k / 24k / 48k audio options                 |
| `speed`       | float  | `1.0`       | Adjust speaking speed (0.5–2.0)                    |
| `consistency` | float  | `0.5`       | Reduces word repetition (Lightning Large only)     |
| `similarity`  | float  | `0.7`       | Match the style of reference audio (cloned voices) |

## **Tips**

- **Ultra fast :** Lightning generates ~10 seconds of speech in ~100 ms → great for real-time apps.
- **Voice Cloning :** Use your cloned `voice_id` and set `model="lightning-large"` for higher similarity.
- **Audio Quality :**
  - 24k → best balance
  - 48k → studio quality
  - 8k → telephony / call center use
