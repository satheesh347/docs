# **How to Use Waves Text-to-Speech (TTS)**

Waves lets you convert any text into **hyper-realistic speech in under 100 ms**.\
This quick guide shows you how to make your first TTS request using cURL, Python, or JavaScript.

## **1. Get Your API Key**

- Log in to your Waves dashboard ‚Üí open the **API Key** tab ‚Üí copy your key.
- For security, store it in an environment variable:

```bash
export SMALLEST_API_KEY="your_api_key_here"
```

## **2. API Endpoint**

```bash
https://waves-api.smallest.ai/api/v1/lightning/get_speech
```

## **3. Required Headers**

```bash
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

## **4. Make Your First Request**

```bash
curl -X POST https://waves-api.smallest.ai/api/v1/lightning/get_speech \
  -H "Authorization: Bearer $SMALLEST_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Welcome to Waves, the fastest text to speech platform.",
    "voice_id": "default",
    "sample_rate": 24000,
    "speed": 1.0
  }' \
  --output speech.wav
```

```python
from smallestai.waves import WavesClient

client = WavesClient(api_key="YOUR_API_KEY")

client.synthesize(
    text="Welcome to Waves, the fastest text to speech platform.",
    voice_id="default",
    sample_rate=24000,
    speed=1.0,
    save_as="speech.wav"
)
```

```javascript
const fs = require('fs');

const response = await fetch(
  'https://waves-api.smallest.ai/api/v1/lightning/get_speech',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.SMALLEST_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      text: 'Welcome to Waves, the fastest text to speech platform.',
      voice_id: 'default',
      sample_rate: 24000,
      speed: 1.0
    })
  }
);

const buffer = await response.arrayBuffer();
fs.writeFileSync('speech.wav', Buffer.from(buffer));
```

## **5. Response Format**

- **Status:** `200 OK`
- **Content-Type:** `audio/wav`
- **Format:** PCM WAV, mono
- The body contains the **raw audio bytes** you can save as a `.wav` file.

## **6. Key Parameters**

| Parameter     | Type   | Default     | What it Does                                       |
| :------------ | :----- | :---------- | :------------------------------------------------- |
| `text`        | string | ‚Äî           | Text you want to convert (recommended ‚â§500 chars)  |
| `voice_id`    | string | `"default"` | Choose a built-in or custom cloned voice           |
| `sample_rate` | int    | `24000`     | 8k / 16k / 24k / 48k audio options                 |
| `speed`       | float  | `1.0`       | Adjust speaking speed (0.5‚Äì2.0)                    |
